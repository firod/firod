Пишем два свитча подряд, нажимаем "применить" и кликер отжирает больше гига оперативы.

Код: Clickermann
  1. SWITCH($var)
  2. CASE(1)
  3. // случай 1
  4.  
  5. case(2)
  6. // случай 2
  7.  
  8. DEFAULT
  9. // случай по умолчанию 
  10.  
  11. END_SWITCH
  12.  
  13. SWITCH($var)
  14. CASE(1)
  15. // случай 1
  16.  
  17. case(2)
  18. // случай 2
  19.  
  20. DEFAULT
  21. // случай по умолчанию 
  22.  
  23. END_SWITCH
  24.  
  25. halt
