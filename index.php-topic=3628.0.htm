Код: Clickermann
  1. $r = RND(1,100)
  2. FOR($a=0,$a<$r)
  3.    LCLICK(100,100)
  4.    WAITMS(50)
  5. END_CYC

Ещё есть вопросик по Рандому. Как прописать чтоб в поле по координатам (допустим (100,100) вписался текст Рандомный. Любые 30 знаков и цифр A-Z и 0-9 .Считывание с txt рандомной строчки с записью в буфер- Не подходит.

Для печати больших букв требуется последняя бета версия.
http://crapware.aidf.org/forum/index.php?topic=356.msg24549#msg24549

Чтобы KEYSTRING работала правильно раскладки кликера и целевого окна должны совпадать.
Код: Clickermann
  1. $l = 30  // длина слова
  2. $s = "ABCDEFGHIJKLMNOPQRSTUVWXYZ1234567890" // Набор символов
  3. $str = ""
  4. FOR($i=0,$i<$l)
  5.    $str = STRCONCAT($str,STRCUT($s,RND(1,STRLEN($s)),1))
  6. END_CYC
  7.  
  8. LCLICK(100,100)
  9. WAITMS(50)
  10. KEYSTRING($str)
  11. WAITMS(50)
  12.  
  13. HALT

Смену раскладки можно сделать автоматически.
Код: Clickermann
  1. $l = 30  // длина слова
  2. $s = "ABCDEFGHIJKLMNOPQRSTUVWXYZ1234567890" // Набор символов
  3. $str = ""
  4. FOR($i=0,$i<$l)
  5.    $str = STRCONCAT($str,STRCUT($s,RND(1,STRLEN($s)),1))
  6. END_CYC
  7.  
  8. LCLICK(100,100) // Клик
  9. WAITMS(50)
  10. // Смена раскладки 1033 En 1049 Ru
  11. SETKBLAYOUT(HGET(100,100), 1033) // Для целевого окна по координате
  12. SETKBLAYOUT($_hwnd_self, 1033) // для кликера
  13. WAITMS(50)
  14. KEYSTRING($str) // Вводим символы
  15. WAITMS(50)
  16.  
  17. HALT  // Стоп
