Ладно. Вот ещё одно задание. Сам давно подумывал сделать и вот только что натолкнулся и вспомнил. Вполне себе бытовое и нужное.
https://www.codewars.com/kata/checking-groups

Гуглоперевод

В принципе всё понятно, проверить правильность вложенности скобок в строке. Выдать ответ "правильно" / "неправильно".

Годная оказалась задача (3 in 1). Одним решением поднял на Codewars целый уровнь. Функция подходит для решения трёх кат разных уровней.
Вот они все: (может еще есть, я не искал, может кому пригодится)
Checking Groups (6 kyu)
Valid Parentheses (5 kyu)
Valid Braces (4 kyu)

Решал эту задачу на JavaScript. Для Clickermann соответственно тоже написал подпрограмму groupCheck($s) + оригинальные тесты из задачи "Checking Groups".

Spoiler:  Ката "Checking Groups" (решение)
Код: Clickermann
  1. #name "Checking Groups" // Clickermann v4.12 (build 001)
  2.  
  3. // Задача "Checking Groups" - https://www.codewars.com/kata/checking-groups
  4.  
  5. //-------------------------------- Test Cases:
  6. //
  7. // Тестовые строки от авторов задачи. Копируем их в массив (вместе с комментариями и прочим "мусором") для проверки своего решения.
  8. // Как оказалось (после выполнения задания), тестовые строки не содержат никаких символов кроме скобок, но мы их добавим.
  9.  
  10. $arr[0] = "Test.expect(groupCheck('()'), 'That group was matched correctly');"
  11. $arr[1] = "Test.expect(groupCheck('({})'), 'That group was matched correctly');"
  12. $arr[2] = "Test.expect(groupCheck('[[]()]'), 'That group was matched correctly');"
  13. $arr[3] = "Test.expect(groupCheck('[{()}]'), 'That group was matched correctly');"
  14. $arr[4] = "Test.expect(groupCheck('([])'), 'That group was matched correctly');"
  15. $arr[5] = "Test.expect(groupCheck('{}([])'), 'That group was matched correctly');"
  16. $arr[6] = "Test.expect(groupCheck('{[{}[]()[]{}{}{}{}{}{}()()()()()()()()]{{{[[[((()))]]]}}}}(())[[]]{{}}[][][][][][][]({[]})'), 'That group was matched correctly');"
  17. $arr[7] = "Test.expect(groupCheck(''), 'That group was matched correctly');"
  18. //
  19. $arr[8] = "Test.expect(!groupCheck('{(})'), 'That group was matched incorrectly');"
  20. $arr[9] = "Test.expect(!groupCheck('[{}{}())'), 'That group was matched incorrectly');"
  21. $arr[10] = "Test.expect(!groupCheck('{)[}'), 'That group was matched incorrectly');"
  22. $arr[11] = "Test.expect(!groupCheck('[[[]])'), 'That group was matched incorrectly');"
  23. $arr[12] = "Test.expect(!groupCheck('()[]{]'), 'That group was matched incorrectly');"
  24. $arr[13] = "Test.expect(!groupCheck('{([]})'), 'That group was matched incorrectly');"
  25. //
  26. $arr[14] = "Test.expect(!groupCheck('[])'), 'That group was left open or prematurely closed');"
  27. $arr[15] = "Test.expect(!groupCheck('([]))'), 'That group was left open or prematurely closed');"
  28. $arr[16] = "Test.expect(!groupCheck('{{))'), 'That group was left open or prematurely closed');"
  29. $arr[17] = "Test.expect(!groupCheck('{}{'), 'That group was left open or prematurely closed');"
  30. $arr[18] = "Test.expect(!groupCheck('{'), 'That group was left open or prematurely closed');"
  31. $arr[19] = "Test.expect(!groupCheck(']'), 'That group was left open or prematurely closed');"
  32. $arr[20] = "Test.expect(!groupCheck('{{{{{{{{{{{((((((([])))))))}}}}}}}}}}'), 'That group was left open or prematurely closed');"
  33. //
  34. $arrsize = ARRSIZE($arr)
  35.  
  36. //-------------------------------- Решение (было бы другим, если бы задание было правильно сформулировано)
  37.  
  38. SUB(groupCheck, $s)
  39.  
  40.    $s = STRFILTER($s, "[({})]", 1)
  41.  
  42.    $check = 1
  43.  
  44.    WHILE( (STRLEN($s) > 0) & ($check > 0) )
  45.       $lastStrLength = STRLEN($s)
  46.       $s = STRREPLACE($s, "()", "")
  47.       $s = STRREPLACE($s, "{}", "")
  48.       $s = STRREPLACE($s, "[]", "")
  49.       $check = $lastStrLength - STRLEN($s)
  50.    END_CYC
  51.  
  52.    IF(STRLEN($s))
  53.       LOGWRITE ("false")
  54.    ELSE
  55.       LOGWRITE ("true")
  56.    END_IF
  57.  
  58.    //   LOGWRITE ( !!STRLEN($s) ) // 0 - true, 1 - false
  59.  
  60. END_SUB
  61.  
  62. //-------------------------------- Тесты
  63.  
  64. LOGCLEAR
  65.  
  66. FOR($i = 0, $i < $arrsize)
  67.  
  68.    groupCheck($arr[$i])
  69.  
  70. END_CYC
  71.  
  72.  
  73. HALT
  74.
